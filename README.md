# DeckNextLevel

Apresentações/decks da **Next Level Tech** publicados como **GitHub Pages** em **marcosgianoni.github.io/DeckNextLevel**.
Cada apresentação é uma página HTML estática servida direto, sem build.

- **URL:** https://marcosgianoni.github.io/DeckNextLevel/ (cada deck em `/<slug>/`)
- **Sem portal na raiz** — cada deck é compartilhado pela URL direta.

## Como funciona

- **Sem build/CI** — o GitHub serve os HTML exatamente como estão na branch `main` (pasta raiz).
- **Domínio nativo** do GitHub Pages (project page `/DeckNextLevel/`) — sem `CNAME`.
- **Marca:** Eclipse (`#080F17`), Voltage Blue (`#3DA3EC`), Stardust (`#A3B0C5`). Logo em `nextlevel-logo.png`.
- Padrão visual, fluxo de publicação e uso da marca: ver [`CLAUDE.md`](CLAUDE.md).
