# DeckNextLevel

Apresentações/decks da **Next Level Tech** publicados como **GitHub Pages**.
Cada apresentação é uma página HTML estática servida direto, sem build.

- **Portal:** https://marcosgianoni.github.io/DeckNextLevel/
- Cada apresentação vive em uma subpasta com `index.html` (o caminho da pasta é a URL).
